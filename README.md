# Rick & Morty Explorer рџ›ё

An interactive single-page application built with **Vite** and the [Rick and Morty API](https://rickandmortyapi.com/). Browse, filter, sort, and save your favourite characters and episodes from across the multiverse.

---

## Features

- **Character browser** вЂ” paginated grid or list view of 800+ characters
- **Episode browser** вЂ” all seasons and episodes with air dates and cast counts
- **Search** вЂ” live debounced search by character name
- **Filters** вЂ” filter by status (Alive/Dead/Unknown), species, and gender
- **Sorting** вЂ” sort by ID (asc/desc) or name (AвЂ“Z / ZвЂ“A)
- **Favourites** вЂ” save characters between sessions via LocalStorage
- **Character detail modal** вЂ” full stats and episode list for each character
- **Dark / Light theme** вЂ” user preference saved to LocalStorage
- **Grid / List view toggle** вЂ” preference saved to LocalStorage
- **Responsive design** вЂ” works on mobile, tablet, and desktop

---

## API

- **Base URL:** `https://rickandmortyapi.com/api`
- **No API key required**
- **Documentation:** https://rickandmortyapi.com/documentation
- **Endpoints used:**
  - `GET /character` вЂ” paginated character list with filter params
  - `GET /character/:id` вЂ” single character detail
  - `GET /character/:ids` вЂ” multiple characters by comma-separated IDs
  - `GET /episode` вЂ” paginated episode list

## Installation

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/rick-and-morty-app.git
cd rick-and-morty-app

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Build for production
npm run build

# 5. Preview production build
npm run preview
```
